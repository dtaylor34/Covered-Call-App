# Covered Calls Manager

Educational paper-trading web app for learning covered call strategies. React + Vite + Firebase (Auth, Firestore, Hosting, Functions) + Stripe.

See `docs/PRD.md` for full product requirements and `docs/LAUNCH_CHECKLIST.md` for setup and deployment.

## Quick start

```bash
npm install
npm run dev
```

For local Firebase emulators: `npm run dev:full` (requires [Firebase CLI](https://firebase.google.com/docs/cli)).
